# tweetoffds15

## Installation 

Download the repo and naviate there from the command line:

```sh
git clone git@github.com:macrotentional/tweetoffds15.git
cd tweetoffds15
```


## Setup

Setup and activate a virtual environment:

```sh
pipenv install
pipenv shell
```

## Usage 

Run the web app:

```sh
FLASK_APP=web_app flask run
```
